# Gerador QRCode PIX

Aplicação desktop para gerar QR Codes PIX válidos.

## Como usar:

### Executar em modo desenvolvimento:
```bash
npm start
```

### Gerar executável para Windows:
```bash
npm run build:win
```

O executável será gerado na pasta `dist/`

### Outros sistemas:
- **macOS**: `npm run build:mac`
- **Linux**: `npm run build:linux`

## Recursos:
- ✅ Geração de QR Code PIX válido
- ✅ Máscara monetária brasileira
- ✅ Validação de campos obrigatórios
- ✅ Cópia do código PIX
- ✅ Interface moderna e responsiva

## Tecnologias:
- Electron
- HTML5 / CSS3 / JavaScript
- QRCode.js
