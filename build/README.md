# Gerador QRCode PIX - Ícone Placeholder

Este diretório deve conter os ícones da aplicação:

- **icon.ico** - Ícone para Windows (256x256 pixels)
- **icon.icns** - Ícone para macOS
- **icon.png** - Ícone para Linux (512x512 pixels)

## Como adicionar ícones:

1. Crie um ícone 512x512 pixels
2. Converta para os formatos necessários:
   - Windows: .ico
   - macOS: .icns
   - Linux: .png

Você pode usar ferramentas online como:
- https://www.icoconverter.com/
- https://cloudconvert.com/

Por enquanto, o Electron usará o ícone padrão.
